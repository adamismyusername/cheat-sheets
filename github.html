Git & GitHub Cheat Sheet

Quick reference for common commands ‚Ä¢ Click any command for details

Setup & Config

git config --global user.name "Your Name"

Set your name for commits

git config --global user.email "you@email.com"

Set your email for commits

git config --list

Show all config settings

git init

Initialize new repo in current folder

git clone <url>

Clone a remote repo locally

Basic Workflow

git status

Check what's changed

git add .

Stage ALL changes

git add <file>

Stage specific file

git commit -m "message"

Commit with message

git commit -am "message"

Add + commit tracked files in one shot

git push

Push commits to remote

git pull

Fetch + merge from remote

Branches

git branch

List local branches

git branch -a

List ALL branches (local + remote)

git branch <name>

Create new branch

git checkout <branch>

Switch to branch

git checkout -b <name>

Create AND switch to new branch

git branch -d <name>

Delete branch (safe)

git branch -D <name>

Force delete branch

Merging

git merge <branch>

Merge branch into current branch

git merge --no-ff <branch>

Merge with commit (no fast-forward)

git merge --abort

Abort a merge with conflicts

git rebase <branch>

Rebase current onto branch

git rebase --abort

Abort a rebase

Remotes

git remote -v

Show remote URLs

git remote add origin <url>

Add remote called "origin"

git push -u origin <branch>

Push and set upstream (first push)

git fetch

Download remote changes (no merge)

git push origin --delete <branch>

Delete remote branch

History & Logs

git log

Show commit history

git log --oneline

Compact one-line log

git log --graph --oneline

Visual branch graph

git diff

Show unstaged changes

git diff --staged

Show staged changes

git show <commit>

Show specific commit details

Undoing Stuff (Be Careful)

git checkout -- <file>

Discard changes to file

git reset HEAD <file>

Unstage a file

git reset --soft HEAD~1

Undo last commit, keep changes staged

git reset --hard HEAD~1

git revert <commit>

git clean -fd

‚ö†Ô∏è Delete untracked files & folders

Stashing

git stash

Stash current changes

git stash save "message"

Stash with description

git stash list

List all stashes

git stash pop

Apply and remove latest stash

git stash apply

Apply stash (keep in stash list)

git stash drop

Delete latest stash

Tags

git tag

List all tags

git tag v1.0.0

Create lightweight tag

git tag -a v1.0.0 -m "msg"

Create annotated tag

git push origin v1.0.0

Push specific tag

git push origin --tags

Push all tags
